ez_climate.tree module¶

class ez_climate.tree.TreeModel(decision_times, prob_scale=1.0)[source]¶

Bases: object

Tree model for the EZ-Climate model. It provides the structure of a non-recombining tree.

Parameters:
  • decision_times (ndarray or list) – years in the future where decisions will be made
  • prob_scale (float, optional) – scaling constant for probabilities
