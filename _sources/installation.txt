Installation
============

Requirements
------------

dlw is compatible with Python 2 and 3.

Numpy_ is required, and we recommend matplotlib_ for visualization of 
results.

.. _Numpy: http://www.numpy.org/
.. _matplotlib: http://www.matplotlib.org/


Install dlw
------------

We encourage you to use pip_ to install dlw on your system. 
::
   pip install dlw

If you wish to build from sources, download_ or clone_ the repository.

.. _download: https://pypi.python.org/pypi/dlw/
.. _clone: https://github.com/oscarsjogren/dlw.git

.. _pip: http://www.pip-installer.org/en/latest/
